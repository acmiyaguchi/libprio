Detailed Description

These functions attempt to implement CCA-secure public-key encryption using the NSS library. We use hashed-ElGamal encryption with Curve25519 as the underlying group and AES128-GCM as the bulk encryption mode of operation.

I make no guarantees that I am using NSS correctly or that this encryption scheme is actually CCA secure. As far as I can tell, NSS does not provide any public-key hybrid encryption scheme out of the box, so I had to cook my own. If you want to be really safe, you should use the NaCl Box routines to implement these functions.
