PrioPacketClient_encode

Takes as input a pointer to an array (data_in) of boolean values whose length is equal to the number of data fields specified in the config. It then encodes the data for servers A and B into a string.

NOTE: The caller must free() the strings for_server_a and for_server_b to avoid memory leaks.
