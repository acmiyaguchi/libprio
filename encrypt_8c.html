SECStatus PublicKey_import ( PublicKey *  pk,
const unsigned char *  data,
unsigned int  dataLen 
)

Import a new curve25519 public/private key from the raw bytes given. When importing a private key, you must pass in the corresponding public key as well. The byte arrays given as input should be of length CURVE25519_KEY_LEN.

These functions will allocate a new PublicKey/PrivateKey object, which the caller must free using PublicKey_clear/PrivateKey_clear.

Import a new curve25519 public/private key from a hex string that contains only the characters 0-9a-fA-F.
